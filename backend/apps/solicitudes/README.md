# solicitudes

Esta app contiene la solicitud principal del negocio.

Que hace:
- Define el numero de solicitud (`CD1`, `CD2`, etc.).
- Mantiene el estado general del proceso.
- Relaciona al solicitante y al asesor.
- Sirve como entidad padre para PRESELECTA, historial, XCORE y decision.

Modelo principal:
- `Solicitud`

Cuando se usa:
- Durante todo el flujo. Es la entidad central del credito digital.
